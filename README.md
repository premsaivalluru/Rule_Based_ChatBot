# 🎓 Smart Student Support Chatbot

A simple Python-based chatbot designed to help students access academic information such as attendance, marks, timetables, courses, and exam schedules through a command-line interface.

---

## 📌 Features

- 📅 View exam schedules (Mid/Sem)
- 🕒 Display branch-wise timetables
- 📚 Show available courses for each branch
- 📊 Check student attendance percentage
- 📝 View subject-wise marks and average score
- 💬 Interactive chatbot conversation system

---

## 🛠️ Technologies Used

- Python
- Pandas

---

## 🏫 Supported Branches

- CSE
- ECE
- ME
- CE
